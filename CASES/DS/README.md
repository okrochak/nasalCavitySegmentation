### different CASE files used with DeepSpeed
Here, the naming convention is Framework_dataset_model_extension\
for example `DS_ATBL_CAE_mod.py` means:\
FrameWork: DeepSpeed (DS) \
Dataset: Actuated TBL (ATBL): `https://b2share.fz-juelich.de/records/5dbc8e35f21241d0889906136cf28d26` \
Model: Convolutional AutoEncoder (CAE) \
Extension: Modified with several code optimisation
